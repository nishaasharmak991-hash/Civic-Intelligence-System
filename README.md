Civic Intelligent system is a citizen complaint management and intelligent decision making platform. It allow citizens to register complaints through three modes: text, voice or image and automatically analyzes them using AI and generate the recommendable actions, root cause, stores complaints in a database with their complaint ids and produces downloadable pdf reports.

It include varios features like:
Text or voice or image complaint registration
Analysis of complaint using Google Gemini
Decision Report
SQLite database storage
PDF report of complaint
Admin dashboard
Complaint status mangement
complaint analytics and stats
complaint search

Technologies used:
Python
Streamlit
Google Gemini
SQLite
Pandas
matplotlib
speech recognition
git & github
